Encastrement d'une prise ou de plusieurs prises

C'est quoi l'encastrement ?

Le mot encrastrement veux dire d'insérer quelque chose dans un objet.
Dans notre cas ca signifie la même chose. L'encastrement dans le placo, béton, faux-plafond c'est insérer nos gaine, nos prises, nos intérrupteurs dans le placo où le béton.
Je vais vous mettre des photos pour que vous comprennez bien.

Les plans

Les plans sont très important, il permet de savoir où mettre les prises. A qu'elle distance de ce mur, à qu'elle hauteur, es que c'est une hotte ?
Je vais vous mettre des photos pour que vous comprennez bien.



Sur le plan il y'a plusieurs sortes de prises:
Les prises de courant 16A, c'est les prises traditionnels utiliser pour les taches courantes (exemples: TV, aspirateur, charger le téléphone).
Les prises de courant 20A, celle-ci sont des prises spécialiser car on alimente avec le four, lave-vaiselle, machine à laver et seche linge.
La sortie de câble 32A, celle-ci est pour la plaque de cuisson.
La prise RJ 45 est pour internet, téléphone, imprimantes etc..
La sortie de câble du chauffe-eau est une prise aussi spécialiser mais pour la chauffe-eau.

Pourquoi il y'a des numéros sur les prises ?
Les numéros representer les prises dans chaque circuit.
Je m'explique, il y'a 5 numéros différents (1, 2, 3, 4, 5). Ce qui veux dire que dans le circuit 1, il y'a 8 prises. Dans le circuit 2, il y'a 6 prises. Ce qui permet de savoir combien de prises il y'a sur un circuit.

L'encastrement dans une dalle :

Tout d'abord, nous devant encastrer toutes les prises dans la dalle de la maison.
La premiere chose à faire c'est de tracer où sont placer les différents prises et le tableaux électrique.
Une fois tracer, on peut alors placer les tiges en férailles pour pouvoir attacher les gaines pour ne pas qu'elle retomber dans la dalle lors du coulage du béton.

On peut voir différents couleurs de gaine.
La couleur verte généralement est une gaine pré-filée avec du RJ et TV dedans.
La couleur bleue généralement est aussi une gaine pré-filée avec 3 file de 2.5MM² dedans.
La couleur grise généralement n'est pas une gaine pré-filée donc on l'utilise pour tout les autres usages. (Thermostat, l'éclairage, l'alimentation du logement etc..)
