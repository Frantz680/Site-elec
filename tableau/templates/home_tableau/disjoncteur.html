Les différents disjoncteurs

Les disjoncteurs

Les disjoncteurs protège le matèriel contre les surcharges et les courts-circuits.
L'intensité peut varier de 2 à 32 A.

Interrupteur différentiel

Les intérrupteurs différentiels sont conçu pour protéger les personnes. Il se coupe dès que l'intensité du qui sort du circuit est plus faible que celle qui rentre.
Il exciste différent intérrupteurs différentiels:

Le type AC
Il est utilsé pour les circuits électriques classiques (prises, éclairage, radiateur, etc...)

Le type A
Il est utilsé pour les circuits qui utilisent du courant à composantes continues(Electronique). Il est obligatoire au minimun sur les plaques de cuisson, les machines à laver et les prises de recharge pour véhicule électrique.

Le type F (anciennement HI, HPI ou ASI)
Il est utilisé pour les appareils sensibles comme le matiériel informatique, le congélateur et l'alarme.

Le délesteur

Lorsque vous dépasser la puissance sur votre vote disjoncteur, il coupe l'alimentation de un ou de plusieurs circuits non prioritaire (convecteur). Cela évite le déclanchement du disjoncteur principal.

Le contacteur jour/nuit

Il permet la mise sous tension durant les plages horaires à tarif réduit. Une horloge programmable remplit le même rôle.

Le télérupteur

Il vous permet de mettre trois ou plus interrupteur sur un circuit de lumières, souvent utilisé pour la cage d'escalier.

Le relais temporisé

Il permet de commander la mise sous et/ou hors tension d'un circuit (éclairage) en fonction d'une durée déterminée.

La minuterie

Commandée par un bouton poussoir, elle permet d'allumer et de s'éteindre automatiquement (le garage, cave) pour une durée déterminée.
