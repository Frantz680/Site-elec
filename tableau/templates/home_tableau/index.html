Les fonctions d'un tableaux électrique.

Un tableau électrique sert à distribuer, contrôler mais aussi protéger tous les circuits de votre habitation.

Les modules de protections.

Les disjoncteurs

Les disjoncteurs protège le matèriel contre les surcharges et les courts-circuits.
L'intensité peut varier de 2 à 32 A.

Interrupteur différentiel

Les intérrupteurs différentiels sont conçu pour protéger les personnes. Il se coupe dès que l'intensité du qui sort du circuit est plus faible que celle qui rentre.
Il exciste différent intérrupteurs différentiels:

Le type AC
Il est utilsé pour les circuits électriques classiques (prises, éclairage, radiateur, etc...)

Le type A
Il est utilsé pour les circuits qui utilisent du courant à composantes continues(Electronique). Il est obligatoire au minimun sur les plaques de cuisson, les machines à laver et les prises de recharge pour véhicule électrique.

Le type F (anciennement HI, HPI ou ASI)
Il est utilisé pour les appareils sensibles comme le matiériel informatique, le congélateur et l'alarme.
